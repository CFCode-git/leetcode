Given an integer, write a function to determine if it is a power of two.

给定一个整数，编写一个函数来判断它是否是 2 的幂次方。

Example 1:

Input: 1
Output: true
Explanation: 20 = 1
Example 2:

Input: 16
Output: true
Explanation: 24 = 16
Example 3:

Input: 218
Output: false
